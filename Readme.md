
# nl

  Convert linebreaks into newline characters (\n)

## Installation

```bash
npm install -g nl
```

## Usage

```bash
$ cat in.txt | nl > out.txt
```

```js
var nl = require('nl')
nl(read('./Readme.md'))
```
